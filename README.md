**SEP LAB Collection**

This is a collection of labs for the Software Engineering Practice (SEP) course in Shanghai Jiao Tong University for the seccond semester of freshman year. The Lab folder contains the completed lab codes, and the Lab Init folder contains the original starting files for each lab, for learning and communication.

**Project Structure**

```
SEP/
├─ README.md
├─ Lab/
│  ├─ lab1/  # Lab1: student/course score manager.
|  └─ lab2/  # Lab2: Y86-64 Simulator.         
└─ Lab Init/ # The Original Starting Files and Instructions for each Lab.
```
